## Stack

- HTML + CSS (Pretendard, JetBrains Mono)
- Playwright (PDF 렌더링)
- GitHub Pages (정적 호스팅)

## Local development

```bash
# 미리보기
open index.html

# PDF 재생성 (소스 디렉토리에서)
cd /Users/kyuwan/resume
npm run build
```

## Deploy

`main` 브랜치에 push하면 GitHub Actions가 자동으로 정적 사이트로 배포합니다.
